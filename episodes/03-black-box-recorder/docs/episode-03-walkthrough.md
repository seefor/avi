# Episode 03 Walkthrough: The Black Box Recorder

Use this as the camera-ready flow for Episode 3.

## 1. Opening Hook

"Avi now has one safe tool and a brain that can explain evidence. The next thing it needs is memory. Not memory like a chatbot. Memory as an evidence trail."

## 2. What We Are Building

```text
Approved Observation -> Recorder Entry -> Report -> Human Review
```

## 3. Why This Matters

What to say:

"If Avi is going to become useful, we need to know what happened after the demo is over. What did Avi review? When did it happen? Did it succeed? What did it summarize?"

## 4. Recorder Fields

Explain the fields:

- Timestamp
- Event
- Target
- Requested action
- Status
- Duration
- Summary
- Error details

## 5. Teaching Point

A brain without evidence is just a confident narrator. Avi needs evidence before it earns more capability.
